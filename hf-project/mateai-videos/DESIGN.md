# MateAI — Visual Identity

## Style Prompt

Tech-forward, clean, futuristic. Dark canvas with bright teal/cyan/neon-green accents echoing the MateAI logo. WhatsApp-inspired chat mockups showing automation in action. Confident, business-oriented, modern SaaS aesthetic. Glassmorphism, subtle glow, grid patterns.

## Colors

- `#050A0A` — Background (deep near-black with teal tint)
- `#0B1414` — Surface (card backgrounds)
- `#00D4AA` — Primary accent (teal, matches logo)
- `#06B6D4` — Secondary accent (cyan)
- `#00FF9C` — Highlight (neon green, sparingly)
- `#F0FDF9` — Primary text (near-white with mint tint)
- `#A1A1AA` — Secondary text (cool gray)
- `#25D366` — WhatsApp green (for WA-specific elements)

## Typography

- **Display / Headlines:** `Space Grotesk`, weight 600, letter-spacing -0.025em
- **Body:** `Inter`, weight 400-600

## Motion

- Entrances: `power3.out`, `expo.out`, `back.out(1.4)` — varied per scene
- Staggers: 100-160ms between siblings
- Durations: 0.5-0.8s for hero elements, 0.3-0.5s for supporting
- No bouncy elastic easings (too playful for this brand)

## What NOT to Do

1. Don't use violet/purple (`#8B5CF6`) or pink (`#EC4899`) — breaks brand
2. Don't use Arial, Roboto, or default sans-serifs — use Space Grotesk/Inter only
3. Don't use full-screen linear gradients — use radial glows or solid + localized glow
4. Don't use emojis larger than 48px — they look cartoonish
5. Don't use more than 3 eases per scene — keep motion disciplined
